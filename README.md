# ISEP-Projet-Algo-et-Programmation
Projet : Création d'un Bi-cluster

# Consignes
You will work in groups. A report (pdf format) and your code are due for May, 30th. The report should contain a description of the technical and strategic choices you made, the problems you encountered, and results you obtained.
Presentations will be set on the 5th and 6th of June. 

The objective of this project is (1) to understand the bi-clustering algorithm which allows a dataset to be clustered according to rows and columns simultaneously, (2) to choose and develop an algorithm in the language of your choice (java, C, or python), and (3) to apply this algorithm on a dataset of your choice.

L'idée généralle est la suivante :  
On part d'une BDD qui possède un ensemble de tables et dans chaque tables différents attributs.  
Dans un premier temps on utilise un cluster afin de créer des groupes qui réunnissent un ensemble de données selon certains paramètres que l'on a fixé. Ex : à partir de notre BDD créer un groupe qui rassemble toutes les personnes de couleur noir  
Puis dans un deuxième temps on peut également créer des groupes selon les attributs des tables. Par ex la couleur de peau d'une personne est reliée à la taille de son s**. On obtient alors un algorithme de bi-clustering car il permet de créer des groupes à partir des données et des attributs.
